# DiscordCustomStatus
Программа для создания кастомных дискорд статусов

# Как установить?

Для запуска нужно 
1. Cкачать архив с файлами

2. Распаковать его 

3. Открыть CMD и перейти в директорию с ней

4. Вписать команду 
``pip3 install -r requirements.txt``

5. Открыть gui.exe

После чего откроется консоль с инструкцией
